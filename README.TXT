PROJECT TITLE: Bingo Game
PURPOSE OF PROJECT: To make a simulation of a bingo game and picking numbers at random until one of the conditions for getting a bingo is reached
VERSION or DATE: Winter 2013/2014
HOW TO START THIS PROJECT: After opening in BlueJ, right-click "BingoGame", then left-click
"void main(String[] Args)", then click "Ok"
AUTHORS: Noah Getz
USER INSTRUCTIONS: After starting the project, change the parameters for getting a bingo, then hit play for the program to start picking random numbers
